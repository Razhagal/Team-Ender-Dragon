Java

Future coffee writing with Java:

Java е по-скоро оракулски език. Използва се предимно за предсказване на бъдещето с кафе.
Също като C# и Java е базиран на C.
Java е език с отворена книга, което означава, че всички магьосници могат да допринасят за описаната в него магия.
(Това се осъществява изключително трудно и е почти невъзможно да ви мине номерът пред Оракула, освен ако не сте голямо магьосническо сборище като International Burst Magic Corporation(IBM))
Java към момента е собственост на Оракула.
Java е по-бавен език за заклинания от C#, но за сметка на това рънва на Виртуална Магия(VM).
Понеже Java може да върви на VM, може да се използва навсякъде: чашки, чинийки, тенджерки, котли и т.н. Стига да имате кафенце под ръка, може да използвате Java.
